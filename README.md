# ATOM 电商物料 · ATOM eCommerce Material

BodyPark **ATOM 口袋 AI 私教机**的电商详情页 / 官网对比图物料。包含**套装对比**与**会员权益对比**两组图，成品 PNG + 可再生成的 HTML 源码。

> 数据与命名以《ATOM 产品定价与套装组合说明》为准。政策更新时请同步本仓库。

## 两组图的用途

| 图 | 版本 | 用途 |
|---|---|---|
| **套装对比** | 中 / 英 | 下单页帮用户在三档 Kit 间选择，突出旗舰满配 |
| **会员对比 · A 矩阵版** | 中 / 英 | 逐项 ✓/— + 配额数值，面向**专业 / 理性对比**用户 |
| **会员对比 · B 叠加版** | 中（英文待补） | 三张会员卡「含前档 + 新增」，面向**电商快速沟通** |
| **会员对比 · C 简洁版** | 中（英文待补） | B 的精简对齐版：横向分带，用量与备注三列同高，最紧凑 |

**建议**：详情页首屏放 **C**（最紧凑、对齐最干净），下方放 **A**（供逐项核对）。B 是 C 的前一版，保留备选。

**C 简洁版的对齐做法**：三列不再各自流式排布，而是拆成横向 band（表头 / 继承说明 / 功能 / 用量 / 即将），同一 band 的三个单元格共用一行，因此**用量框与备注在三列中落在同一水平线**；用量统一为「对话 / 实时 / 存储」三项指标，Basic 缺的用 — 占位，做到逐行对齐。

## 设计系统

**套装图**：单一绿色，三档**渐进浅→深**色带，表达"越往上越超值"。

**会员图**：三档**三色区分**（本质是三种不同的东西，不是同一维度的递进）——

| 档 | 列色 | 黑卡标签文字 |
|---|---|---|
| Basic 免费版 | 灰 | 银灰字 |
| Plus 进阶会员 | 绿 | 亮绿字 |
| Pro 专业会员 | 黑金 | 金字 |

共通规范：字号 **3 档**（20 / 13.5 / 9.5）、勾选为粗线条深色（非实心块，避免卡通感）、**极简分隔线**（仅表头、分组之间、表尾）、**直角**无圆角（贴合运动产品调性）、纯白底无边框（便于嵌详情页）。

## 快速上手

- **只要图**：直接用 `assets/` 下的 PNG（@2x、白底、手机屏宽）。
- **改数据 / 出新图**：编辑 `charts/*.html` 顶部的 `DATA` 对象，再按 [`docs/regenerate.md`](docs/regenerate.md) 用无头 Chrome 重新导出。
